centos7学习(一)--centos7安装和配置

centos7安装和配置

1.前期准备

1.1 安装虚拟机

  • VMware-workstation14下载地址: http://download3.vmware.com/software/wkst/file/VMware-workstation-full-14.1.3-9474260.exe
  • 安装步骤”下一步“即可.

1.2 安装centos7

  • centos7下载地址: http://repo.virtualhosting.hk/centos/7/isos/x86_64/CentOS-7-x86_64-DVD-1804.iso
  • 安装步骤”下一步“即可.

2.查看虚拟机中centos7的IP

  1. 虚拟机中登录操作系统 用户名root 密码 root.
  2. 输入IP查询命令 : ip addr,如果发现 ens33 没有 inet 这个属性，那么就没法通过IP地址连接虚拟机.
  3. 查看ens33网卡的配置： vi /etc/sysconfig/network-scripts/ifcfg-ens33 注意vi后面加空格.

  1. 将ONBOOT=no 改为 yes.
  2. 然后重启网络服务：sudo service network restart.
  3. 输入 ip addr.

  • 由图可知,IP地址为:192.168.43.128

1.3 安装XShell

  • Xshell就是一个远程控制Centos的软件,详细介绍请看 百度百科.
  • 下载地址: https://cdn.netsarang.net/63b77b13/Xshell-6.0.0098p.exe
  • 安装步骤”下一步“即可.
  • 连接虚拟机:
    1. 点新建连接.
    2. 输入名称，主机地址 协议 ssh 端口 22 不用改.
    3. 输入用户名,密码
    4. 点连接即可操作虚拟机.

1.4 安装vsftp服务

  • 我们需要向centos操作系统的服务器上上传文件或者下载文件，这时候，ftp有必要安装下.
  1. 安装 vsftp : yum install -y vsftpd
  2. 设置开机启动 : systemctl enable vsftpd
  3. 启动ftp服务 : systemctl start vsftpd.service

  4. 打开防火墙 :

  5. 修改vsftp配置文件,禁止匿名登录 vi /etc/vsftpd/vsftpd.conf

    • 把：anonymous_enable=YES 改为： anonymous_enable=NO输入—> :wq! 保存退出.
  6. 验证:使用 Xftp:
    • Xftp下载地址 https://cdn.netsarang.net/63b77b13/Xftp-6.0.0092p.exe
